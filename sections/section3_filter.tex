\subsection{絞り込み機能}
絞り込み機能は，以下のような機能から構成されている．

\begin{itemize}
  \item キーワード検索機能
  \item ジャンル絞り込み機能
  \item 期間絞り込み機能
\end{itemize}

\subsubsection{キーワード検索機能}
検索窓を利用して，入力したキーワードに応じて情報を検索・絞り込みできる機能．「地図」を選択すると，検索窓が表示される．自身が検索したいキーワードを入力すると，検索したキーワードが含まれる投稿のピンのみが地図上に表示される．検索窓の下にキーワードが含まれる投稿のみが一覧で表示される．

\begin{figure}[H]
        \centering
        \includegraphics[scale=0.5]{sections/fig_filter_general/filter_gen1.jpg}
        \caption{キーワード検索機能}
        \label{fig:Q14}
\end{figure}

\subsubsection{ジャンル絞り込み機能}
特定のジャンルに属するピンのみを表示する機能．「地図」を選択すると，検索窓の下にジャンル選択ボタンが表示される．そのボタンを押してジャンルを選択すると，選択したジャンルの投稿のピンのみが地図上に表示される．ジャンル選択ボタンの下に選択したジャンルの投稿のみが一覧で表示される．

\begin{figure}[H]
        \centering
        \includegraphics[scale=0.5]{sections/fig_filter_general/filter_gen2.jpg}
        \caption{ジャンル絞り込み機能}
        \label{fig:Q15}
\end{figure}


\subsubsection{期間絞り込み機能}
指定した日付または期間内に投稿されたピンのみを表示する機能．「地図」を選択すると，検索窓の下に期間選択ボタンが表示される．そのボタンを押して期間を選択すると，選択した期間の投稿のピンのみが地図上に表示される．期間選択ボタンの下に選択した期間の投稿のみが一覧で表示される．

        \caption{期間絞り込み機能}
